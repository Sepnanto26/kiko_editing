Kiko Editing 
Deskripsi Kiko Editing adalah alat sederhana untuk mengedit video dan gambar secara cepat dan efisien. Proyek ini dirancang untuk pengguna pemula hingga menengah yang ingin melakukan pengeditan dasar seperti pemotongan, penggabungan, dan penambahan efek tanpa perangkat lunak berat.
Fitur Utama
Pemotongan Video: Potong bagian video yang tidak diinginkan dengan mudah.
Penggabungan Klip: Gabungkan beberapa video menjadi satu file.
Penambahan Efek: Tambahkan filter sederhana seperti blur, brightness, dan color correction.
Ekspor Cepat: Ekspor hasil edit dalam format MP4 atau JPG dengan kualitas tinggi.
Antarmuka Sederhana: UI intuitif yang mudah digunakan tanpa pelatihan khusus.
Persyaratan Sistem
OS: Windows 10+, macOS 10.15+, atau Linux (Ubuntu 18.04+).
RAM: Minimal 4GB.
Penyimpanan: 500MB ruang kosong untuk instalasi.
